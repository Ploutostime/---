# 智学秘伴需求文档

## 1. 系统概述

### 1.1 系统名称
智学秘伴

### 1.2 系统描述
一个智能化的学生事务管理与学习效率提升系统，帮助学生高效收纳和整理各类学习事务，生成个性化时间表，并通过智能提醒引导学生快速进入学习状态。系统还提供私人终端服务，根据用户专业推送适配的bilibili学术视频，实现个性化学习内容推荐。

## 2. 核心功能

### 2.1 学生事务管理
- 竞赛日程记录与提醒
- 作业安排整理与截止时间管理
- 时间管控功能，设置学习时长目标
- 时间协调，自动安排学习计划
- 考研考级相关信息管理
- 日程规划与任务优先级设定
- 文件夹管理功能，支持创建和组织各类学习资料文件夹

### 2.2 时间表生成
- 自动生成个性化时间表
- 将时间表置顶显示于手机、iPad、计算机屏幕正上方
- 实时更新和同步跨设备显示

### 2.3 智能提醒功能
- 手机开启时自动浮窗显示bilibili收藏的学习课程
- 学习状态智能识别和提醒
- 快速进入学习状态的引导提示
- 私人终端专业匹配的bilibili学术视频推送
- 根据用户专业标签个性化推荐学术视频内容

### 2.4 效率提升工具
- 学习时间统计和分析
- 学习习惯养成提醒
- 知识点收藏和复习提醒

### 2.5 私人终端视频推荐
- 用户专业信息采集和标签化
- 基于专业领域的bilibili学术视频筛选
- 个性化视频内容推送
- 视频观看历史和偏好记录

## 3. 设计风格

### 3.1 配色方案
主色调采用深蓝色(#1E3A8A)与绿色(#10B981)搭配，营造专业而富有活力的学习氛围

### 3.2 视觉细节
采用现代扁平化设计，8px圆角处理，柔和阴影效果增强层次感，线性图标风格保持简洁统一

### 3.3 整体布局
左侧导航栏+主内容区域的经典布局，时间表采用表格式设计便于查看，事务管理使用卡片式布局便于分类整理，私人终端采用瀑布流或卡片式展示视频内容
