## 技术实现

### 图片来源

所有动漫人物形象图片都通过专业的图片搜索服务获取，确保：
- 高质量的动漫风格
- 符合形象特点
- 视觉效果出色
- 版权合规

### 图片存储

**存储方式：**
- 图片存储在 CDN 服务器
- 使用完整的 URL 路径
- 支持快速加载
- 自动缓存优化

**数据库字段：**
```sql
-- secretary_avatars 表
avatar_url TEXT  -- 存储图片的完整 URL
```

### 图片显示

**React 组件实现：**

```typescript
// 获取形象图片
const getAvatarImage = () => {
  if (!config?.avatar || !config.avatar.avatar_url) return null;
  return config.avatar.avatar_url;
};

// 显示图片
{getAvatarImage() ? (
  <img 
    src={getAvatarImage()!} 
    alt={config.name}
    className="w-full h-full object-cover"
  />
) : (
  <span className="text-2xl">{getAvatarIcon()}</span>
)}
```

**样式处理：**
```css
/* 圆形头像容器 */
.avatar-container {
  width: 64px;
  height: 64px;
  border-radius: 50%;
  overflow: hidden;
  border: 2px solid hsl(var(--primary) / 0.2);
}

/* 图片填充 */
.avatar-image {
  width: 100%;
  height: 100%;
  object-fit: cover;
}
```

### 性能优化

**加载优化：**
- 使用 `loading="lazy"` 延迟加载

**显示优化：**

## 常见问题

### Q1: 如何更换秘书形象？

A: 进入设置页面 → 专属学习秘书 → 形象标签 → 点击想要的形象卡片 → 保存配置

### Q2: 形象图片加载慢怎么办？

A: 图片存储在 CDN 上，通常加载很快。如果网络较慢，请稍等片刻。图片会自动缓存，下次加载更快。

### Q3: 可以自定义上传形象图片吗？

A: 目前暂不支持自定义上传。我们提供了6种精心挑选的动漫形象供选择。未来版本会考虑添加自定义功能。

### Q4: 形象图片会占用很多流量吗？

A: 不会。每张图片都经过优化，大小适中。而且图片会被浏览器缓存，只需加载一次。

### Q5: 形象图片在哪些地方显示？

A: 形象图片会在以下位置显示：
1. 设置页面的形象选择卡片
2. 仪表盘的秘书问候卡片
3. 浮窗秘书（展开和最小化状态）

### Q6: 不同形象的图片风格一致吗？

A: 是的。所有形象都采用统一的动漫风格，确保视觉一致性和美观度。

### Q7: 形象图片会定期更新吗？

A: 目前的形象是精心挑选的经典动漫风格。未来可能会根据用户反馈和节日主题更新形象。

### Q8: 形象图片版权有问题吗？

A: 所有形象图片都通过正规渠道获取，确保版权合规。用户可以放心使用。
