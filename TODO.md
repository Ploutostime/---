## 📝 最新更新

### 2025-12-17 - AI生成秘书3D形象
- 创建 Edge Function `generate-secretary-3d-avatar`
- 集成文生图API，支持3D形象生成
- 为每个秘书类型定制专属prompt（萝莉、御姐、大叔、霸总等15种）
- 在管理员页面添加"秘书形象管理"标签页
- 支持批量生成所有15个秘书的3D形象
- 支持单个秘书形象重新生成
- 自动更新数据库中的avatar_url字段

### 使用方法
1. 使用管理员账号登录系统
2. 进入"管理员控制台"页面
3. 切换到"秘书形象管理"标签页
4. 点击"批量生成"按钮生成所有秘书的3D形象
5. 或点击单个秘书的"重新生成"按钮单独生成
